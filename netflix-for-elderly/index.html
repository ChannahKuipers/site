Netflix for Elderly

For the course Big Information Architecture, there had to be an improvement on the user interface of the Netflix website for elderly people. How to improve Netflix to make the interface more suited for elderly (75+)?
There had to be a thought information architecture, so the target group can easily navigate trough the website.

This project was an assignment for the subjects Big IA during the international minor User Experience Design.

Learnings:
Creating Sitemaps, Content Models, Flowcharts and Wireframes.
